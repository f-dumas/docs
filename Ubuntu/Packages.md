Packages
======

### Packages

#### Ubuntu version
```
lsb_release -a
```

Other details:

```
cat /proc/version
```

#### List of available packages

```
sudo apt-get update
sudo apt-cache pkgnames | grep php7.1
```

#### Manually install a software

```
mv my_software /opt/
chmod a+x /opt/my_software
```

Add to main dashboard:
```
Go to ~/.local/share/applications 
and create your .desktop files there.
```
